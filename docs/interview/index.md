# 面试题库

面试题库用于把知识笔记转化成可表达的答案。建议每个答案都包含结论、原理、场景、边界和代码示例。

## 高频题分类

| 分类 | 示例问题 |
| --- | --- |
| JavaScript | Promise 原理、闭包、原型链、事件循环 |
| 浏览器 | 渲染流程、缓存策略、跨域、安全攻击 |
| Vue | 响应式原理、diff、组件通信、性能优化 |
| 工程化 | Vite 插件、Tree Shaking、代码分割、CI |
| 项目 | 需求拆解、技术选型、性能收益、故障处理 |

## 答案结构

1. 先给结论，控制在 1 到 2 句话。
2. 解释底层机制或关键流程。
3. 说明业务场景中怎么用。
4. 补充边界条件和常见误区。
5. 如果是项目题，补充数据结果和个人职责。

## STAR 项目口播

| 部分 | 内容 |
| --- | --- |
| Situation | 项目背景、业务目标、问题规模 |
| Task | 我的职责、约束条件、验收标准 |
| Action | 技术方案、关键实现、协作过程 |
| Result | 量化结果、复盘收获、后续优化 |

## 练习记录

- 每天选择 3 道题口播录音。
- 每周整理 1 个项目追问专题。
- 对答不顺的问题回链到知识库补笔记。
- 对高频错题增加“反问点”和“边界条件”。
