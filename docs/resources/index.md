# 资源导航

资源导航用于集中管理常用文档、课程、工具和素材。添加资源时建议写一句“为什么值得看”，避免只堆链接。

## 官方文档

| 资源 | 用途 |
| --- | --- |
| MDN Web Docs | 查询 Web API、HTML、CSS、JavaScript 标准行为 |
| Vue Docs | 学习 Vue 语法、组合式 API、生态实践 |
| Vite Docs | 查询开发服务器、构建配置和插件能力 |
| TypeScript Handbook | 学习类型系统、泛型和工程配置 |

## 工程工具

| 工具 | 用途 |
| --- | --- |
| Chrome DevTools | 性能分析、网络调试、内存排查 |
| Lighthouse | 页面质量和性能审计 |
| Vitest | 单元测试与组件测试 |
| Playwright | 端到端测试与页面回归 |

## 资料筛选规则

1. 优先官方文档和一手资料。
2. 课程和文章必须能转化成自己的笔记。
3. 收藏后 7 天内不看就删除或归档。
4. 对同一主题只保留 2 到 3 个高质量入口。
