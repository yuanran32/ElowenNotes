# CSS 高频八股

CSS 面试通常围绕盒模型、格式化上下文、布局方案、层叠规则、响应式和性能展开。回答时不要只背属性，要说明为什么这样写能稳定布局。

## 高频题地图

| 主题 | 必会问题 | 关注点 |
| --- | --- | --- |
| [布局基础](/frontend/bagu/css/layout) | 盒模型、水平垂直居中、多列布局 | 尺寸计算、定位、溢出 |
| [Flex/Grid](/frontend/bagu/css/flex-grid) | Flex 和 Grid 怎么选 | 主轴交叉轴、二维布局 |
| [BFC 与层叠](/frontend/bagu/css/bfc) | BFC 能解决什么问题 | 外边距折叠、清浮动、隔离 |
| [响应式](/frontend/bagu/css/responsive) | 移动端适配怎么做 | viewport、媒体查询、rem、vw |

## 答题顺序

1. 先确认布局目标：一维还是二维、固定还是自适应。
2. 说明核心属性：尺寸、轴线、定位、层叠上下文。
3. 给最小代码：让面试官看到你能落地。
4. 补边界：内容溢出、长文本、兼容性、性能。
5. 关联工程：组件库、移动端、复杂表格或后台页面。
